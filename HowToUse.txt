Compile and run "Driver.java" to start the application.

--------------------------------------------------------------------------------
MovieList.txt
--------------------------------------------------------------------------------
-This contains the total list of movies.
-The order of the movies doesn't matter (list gets sorted).
-You can add movies by following this format:

        MovieName|HaveValue|WantValue|
--------------------------------------------------------------------------------



--------------------------------------------------------------------------------
PurchaseOrder.txt
--------------------------------------------------------------------------------
-Will only make a purchase order if movies are out-of-stock.
 In other words, it will only order more movies if have value is 0.
--------------------------------------------------------------------------------



--------------------------------------------------------------------------------
ShipmentInfo.txt
--------------------------------------------------------------------------------
-This file is written by the store clerk before hand.
-Contains info about new shipment arriving with movie title and its have value.
-Format for file:

        MovieName|HaveValue|
--------------------------------------------------------------------------------
